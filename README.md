## ¿Cómo utilizarlo?

* Clonar el proyecto
* Instalar NodeJS: [Node Js Oficial](http://nodejs.org)
* Instalar http-server
`npm install http-server -g`
* Ir a Directorio / y ejecutar servidor
`http-server ./`
* Abrir navegador 
Dirección: `http://localhost:8080`
